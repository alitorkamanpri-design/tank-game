بخش ۱: آموزش کامل تگ‌های HTML

تگ <h1>

توضیح: این تگ برای تعریف مهم‌ترین عنوان در صفحه استفاده می‌شود.

● ● ● HTML 
<!DOCTYPE html>
<html lang="fa" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>مثال h1</title>
</head>
<body>
    <h1>این مهم‌ترین عنوان صفحه است.</h1>
</body>
</html>

خروجی زنده:

این مهم‌ترین عنوان صفحه است.

بخش ۲: تمرین‌های عملی

در این بخش، تگ‌ها را در قالب یک ساختار کامل HTML تمرین می‌کنید.

تمرین: ساخت لینک

چالش: یک لینک به وبسایت گوگل بسازید که در بدنه (body) یک فایل HTML کامل قرار بگیرد.

● ● ● HTML Template 
<!DOCTYPE html>
<html lang="fa">
<head>
    <meta charset="UTF-8">
    <title>تمرین لینک</title>
</head>
<body>
    <!-- کد خود را اینجا وارد کنید -->
    <a href="https://www.google.com">لینک گوگل</a>
    <!-- این خط برای نمایش جایگاه است -->
</body>
</html>

خروجی زنده:

لینک گوگل

بخش ۳: آزمایشگاه HTML (اجرای زنده)

کد خود را در ویرایشگر زیر وارد کنید و روی "اجرا" کلیک کنید تا خروجی را بلافاصله ببینید.

● ● ● HTML Editor 

<!DOCTYPE html>
<html lang="fa">
<head>
    <meta charset="UTF-8">
    <title>Playground</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>کد خود را اینجا بنویسید.</p>
</body>
</html>
