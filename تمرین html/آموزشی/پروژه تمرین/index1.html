دسته بندی تگ ها

تگ های متن تگ های لینک تگ های لیست رسانه جدول فرم تگ های معنایی

تگ های متن

تگ h1

برای عنوان اصلی صفحه استفاده می شود.

<h1>عنوان اصلی</h1>

عنوان اصلی

تگ p

برای نوشتن پاراگراف.

<p>این یک متن پاراگراف است</p>

این یک متن پاراگراف است

تگ strong

برای تاکید روی متن.

<strong>متن مهم</strong>
متن مهم

تگ em

برای تاکید معنایی روی متن.

<em>متن تاکید شده</em>
متن تاکید شده

تگ های لینک

تگ a

برای ساخت لینک استفاده می شود.

<a href="https://google.com">رفتن به گوگل</a>
رفتن به گوگل

تگ های لیست

تگ ul

لیست نامرتب.

<ul> <li>سیب</li> <li>موز</li> </ul>
  • سیب
  • موز

تگ ol

لیست مرتب.

<ol> <li>مرحله اول</li> <li>مرحله دوم</li> </ol>
  1. مرحله اول
  2. مرحله دوم

تگ های رسانه

تگ img

برای نمایش تصویر.

<img src="https://picsum.photos/200">

تگ video

برای نمایش ویدیو.

<video controls width="250"> <source src="movie.mp4"> </video>

تگ های جدول

تگ table

برای ساخت جدول.

<table border="1"> <tr> <th>نام</th> <th>سن</th> </tr> <tr> <td>علی</td> <td>20</td> </tr> </table>
نام سن
علی 20

تگ های فرم

تگ form

برای دریافت اطلاعات کاربر.

<form> <input placeholder="نام"> <button>ارسال</button> </form>

تگ های معنایی

تگ header

بخش هدر صفحه.

<header> <h2>هدر سایت</h2> </header>
