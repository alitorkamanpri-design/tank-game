# 💼 سیستم مدیریت اضافه‌کاری - نسخه 2.0

## 📋 معرفی

برنامه‌ای کاملا responsive و modern برای مدیریت و ثبت اضافه‌کاری‌های کاری شما.

### ✨ ویژگی‌ها

✅ **ذخیره‌سازی دائمی**: تمام داده‌ها در `localStorage` ذخیره می‌شوند  
✅ **بدون نیاز به Backup**: اطلاعات بعد از بستن برنامه محفوظ می‌ماند  
✅ **Responsive Design**: کاملا سازگار با موبایل، تبلت و لپ‌تاپ  
✅ **تاریخ شمسی**: پشتیبانی کامل از تقویم شمسی  
✅ **صادرات Excel**: دانلود گزارش به صورت Excel  
✅ **فیلترها**: برحسب ماه و سال فیلتر کنید  
✅ **محاسبات خودکار**: ساعات و درآمد خودکار محاسبه می‌شود

## 🎯 مراحل کار

### 1️⃣ ثبت رکورد جدید

- روی دکمه تاریخ کلیک کرده و تاریخ شمسی را انتخاب کنید
- ساعت شروع و پایان را وارد کنید
- مبلغ دستمزد (ریال/ساعت) را وارد کنید
- دکمه **ثبت رکورد** را بزنید

### 2️⃣ مشاهده گزارش

- تمام رکوردها در جدول گزارش نمایش داده می‌شوند
- مجموع ساعات و درآمد خودکار محاسبه می‌شود

### 3️⃣ فیلتر کردن

- ماه و سال را انتخاب کنید
- جدول بر حسب انتخاب شما فیلتر می‌شود

### 4️⃣ ویرایش یا حذف

- دکمه ✏️ برای ویرایش
- دکمه 🗑️ برای حذف

### 5️⃣ صادرات Excel

- دکمه **📊 دانلود Excel** را کلیک کنید
- فایل Excel خودکار دانلود می‌شود
