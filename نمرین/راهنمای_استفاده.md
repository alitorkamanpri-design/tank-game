## 🚀 نحوه اجرا

### **روش 1: استفاده از فایل BAT (راحت‌ترین)**

1. فایل `run_server.bat` را دابل کلیک کن
2. یک پنجره Command Prompt باز می‌شود
3. سرور برروی `http://localhost:8000` راه‌اندازی می‌شود
4. مرورگر خودکار باز می‌شود

### **روش 2: اجرای دستی (Python)**

```bash
cd path/to/project
python server.py
```

سپس در مرورگر بنویس:

- برنامه: `http://localhost:8000`
- بازی‌ها: `http://localhost:8000/games.html`

### **روش 3: بدون Python**

اگر Python ندارید، می‌توانید فایل `index.html` را مستقیماً در مرورگر باز کنید:

- کلیک راست روی `app.html` → "Open with" → مرورگر دلخواه

## 📖 راهنمای استفاده

### **1️⃣ ثبت رکورد جدید**

- تاریخ را انتخاب کن
- ساعت شروع و پایان را وارد کن
- مبلغ دستمزد (ریال/ساعت) را بنویس
- دکمه "✓ ثبت رکورد" را کلیک کن

### **2️⃣ مشاهده داشبورد**

- مجموع ساعات کارکرد ماه جاری
- مجموع درآمد در ریال
- تعداد روزهای اضافه‌کاری

### **3️⃣ استفاده از فیلترها**

- ماه شمسی را انتخاب کن (اختیاری)
- سال شمسی را انتخاب کن (اختیاری)
- جدول خودکار بروزرسانی می‌شود

### **4️⃣ ویرایش رکورد**

- روی دکمه ✏️ کلیک کن
- اطلاعات در فرم لوڈ می‌شود
- تغییرات را انجام بده و دوباره ثبت کن

### **5️⃣ حذف رکورد**

- روی دکمه 🗑️ کلیک کن
- تایید و رکورد حذف می‌شود

### **6️⃣ صادرات Excel**

- دکمه "📊 دانلود Excel" را کلیک کن
- فایل خودکار دانلود می‌شود

## 🛠️ حل مشکلات

### **مشکل: "Address already in use"**

راه‌حل:

```bash
python server.py 8001
```

سپس از `http://localhost:8001` استفاده کن

### **مشکل: Python پیدا نشد**

- Python را نصب کن: https://www.python.org/downloads/
- در نصب آپشن "Add Python to PATH" را انتخاب کن

### **مشکل: داده‌ها محفوظ نمی‌شود**

- اطمینان دار که localStorage مرورگر فعال است
- اگر Private/Incognito Mode هستی، خاموش کن
